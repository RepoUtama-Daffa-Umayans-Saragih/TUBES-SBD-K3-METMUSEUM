## 🆘 Troubleshooting

### CSS Not Loading?
```bash
npm run build
# Then refresh browser (Ctrl+Shift+Del to clear cache)
```

### Database Migration Error?
```bash
# Check database connection in .env
php artisan migrate:status
# If needed, rollback and retry:
php artisan migrate:rollback
php artisan migrate
```

### Charts Not Showing?
```
1. Check browser console (F12)
2. Verify Chart.js CDN is loaded
3. Check canvas element IDs match
4. Verify data is being passed to view
```

### Images Not Uploading?
```bash
# Create storage symlink
php artisan storage:link

# Verify storage directory exists
ls -la storage/app/public/artworks/
```

**For more troubleshooting**, see [QUICK_START_GUIDE.md](QUICK_START_GUIDE.md#-troubleshooting)

## 💡 Common Tasks

### Search Transactions
1. Go to Transactions tab
2. Enter order ID or customer name in search
3. Click Apply or press Enter
4. Results filter automatically

### Export Transactions as CSV
1. Go to Transactions tab
2. Apply filters if needed (status, date)
3. Click "Export to CSV" button
4. File downloads: `transactions_YYYY-MM-DD_HHmmss.csv`

### Add New Artwork
1. Go to Artworks tab
2. Click "Add Artwork" button
3. Fill form (title, department required)
4. Upload images (multi-file)
5. Click Save

### Edit Artwork
1. Go to Artworks tab
2. Find artwork in grid/list
3. Click Edit icon
4. Modify details
5. Add more images if needed
6. Save changes

### View Artwork Details
1. Go to Artworks tab
2. Click artwork card or View button
3. Modal shows full details
4. Click close to dismiss
